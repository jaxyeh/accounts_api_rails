    Introduction

    Here below is a basic API documentation guide for the DistilNetworks Technical Assessment code. We will call this API service as “Account API”, where you can get account names along with its attached IP Addresses and domain names.

    This API documentation was written manually and then created with Slate.

    There is no User Authentication implemented for this example code.

    Accounts

    Get All Accounts

    curl "http://localhost:3000/api/accounts"

    The above command returns JSON structured like this:

    [
    {
        "id": 1,
        "name": "hello",
        "created_at": "2014-10-23T18:36:42.706Z",
        "updated_at": "2014-10-23T18:36:42.706Z"
    },
    {
        "id": 2,
        "name": "helloe",
        "created_at": "2014-10-23T18:45:28.470Z",
        "updated_at": "2014-10-23T18:45:28.470Z"
    }
]

    This endpoint retrieves all accounts.

    HTTP Request

    GET http://example.com/accounts

    HTTP Response

    Parameter type Description
    id integer Primary key ID for each entries
    name string Identifier name for the account holder
    created_at ISO Date Date & time when entry was last created
    updated_at ISO Date Date & time when entry was last updated

    Add Account

    curl -X POST -H "Content-Type: application/json" -d '{"name":"xyz"}' http://localhost:3000/api/accounts

    The above command returns JSON structured like this:

    {
    "id": 2,
    "name": "xyz",
    "created_at": "2014-10-23T18:36:42.706Z",
    "updated_at": "2014-10-23T18:36:42.706Z"
}

    This endpoint creates new account

    HTTP Request

    POST http://example.com/accounts

    Query Parameters

    Parameter Default Description
    name string Identifier name for the account holder

    HTTP Response

    200 OK

    Parameter type Description
    id integer Primary ID
    name string Identifier name for the account holder
    created_at ISO Date Date & time when entry was last created
    updated_at ISO Date Date & time when entry was last updated

    Update Account

    curl -X PUT -H "Content-Type: application/json" -d '{"name":"abc"}' http://localhost:3000/api/accounts/4

    The above command returns JSON structured like this:

    {
    "id": 4,
    "name": "abc",
    "created_at": "2014-10-23T18:36:42.706Z",
    "updated_at": "2014-10-23T18:36:42.706Z"
}

    Update information to existed account

    HTTP Request

    PUT http://example.com/accounts/<id>

    URL Parameters

    Parameter Description
    id The ID of the account to update

    Query Parameters

    Parameter Default Description
    name string Identifier name for the account holder

    HTTP Response

    200 OK

    Parameter type Description
    id integer Primary key ID
    name string Identifier name for the account holder
    created_at ISO Date Date & time when entry was last created
    updated_at ISO Date Date & time when entry was last updated

    Delete Account

    curl -X DELETE -H "Content-Type: application/json" http://localhost:3000/api/accounts/4

    Update information to existed account

    HTTP Request

    PUT http://example.com/accounts/<id>

    URL Parameters

    Parameter Description
    id The ID of the account to update

    204 No Content

    This endpoint retrieves all domains.

    HTTP Request

    GET http://example.com/domains

    HTTP Response

    Parameter type Description
    id integer Primary key ID for each entries
    account_id integer foreign key ID to Account holder
    hostname string Domain Hostname
    origin_ip_address string IPv4 origin IP Address
    created_at ISO Date Date & time when entry was last created
    updated_at ISO Date Date & time when entry was last updated

    Errors

    The Account API uses the following error codes:

    Error Code Meaning
    400 Bad Request
    404 Not Found – The specified request could not be found, or record is not found
    422 Unprocessable Entity - Usually validation error, please see the error message.
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarily offline for maintenance. Please try again later.
